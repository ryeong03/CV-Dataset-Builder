# CV-Dataset-Builder

Say goodbye to noisy data. An intelligent scraper that automatically collects, filters, and deduplicates images for high-performance CV.

---

## 왜 이 프로젝트를 만들었는가 / Why This Project

### 

인턴 업무 중 **교육용 데이터셋**을 수집해야 하는 일이 생김. 다양한 학습자 대상을 위한 이미지 세트를 구축하고, 그 위에 학습 자료를 만들어야 했는데, 일반적인 웹 크롤링만으로는 **품질이 떨어짐**. 노이즈가 많고, 중복이 많고, 학습에 적합하지 않은 이미지가 많이 섞여 들어와서, 그대로 쓰기엔 부담이 컸음. 그래서 **수집·필터링·중복 제거**까지 한 번에 처리할 수 있는, CV(Computer Vision)용 고품질 데이터셋 빌더를 직접 만들고자 함함.

During an internship, I had to collect **educational datasets**—building image sets for various learner audiences and creating learning materials on top of them. With **general-purpose web crawling alone**, the resulting data **quality was too low**: lots of noise, duplicates, and images unsuitable for training. To avoid that, I built this tool—a dataset builder for CV (Computer Vision) that handles **collection, filtering, and deduplication** in one place, so we can get cleaner, more usable image sets for education and model training.
